Marines
=================

Play here: http://ken-soft.com/marines/

Controls:
Primary Weapon: Mouse Click (Primary);
Secondary Weapon: Mouse Click (Secondary);

Movement:
Move Left: Left, A
Move Right: Right, D
Move Up: Up, W
Move Down: Down, S

Enter: Reset
