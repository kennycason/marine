Marines
=================

Play here: http://kennycason.com/marine/

### Controls

| Action | Control |
|--------|---------|
| Primary Weapon | Mouse Click (Primary) |
| Secondary Weapon | Mouse Click (Secondary) |
| Move Left | Left, A |
| Move Right | Right, D |
| Move Up | Up, W |
| Move Down | Down, S |
| Enter | Reset |
